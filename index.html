Projeto Integrador I

Engenharia Mecânica - Unioeste

Introdução

A disciplina de Projeto Integrador I faz parte do curso de Engenharia Mecânica da Universidade Estadual do Oeste do Paraná (Unioeste). Esta disciplina tem como objetivo principal integrar os conhecimentos adquiridos ao longo do curso, proporcionando aos alunos a oportunidade de aplicar na prática os conceitos teóricos estudados em sala de aula.

Durante o curso, os alunos são desafiados a trabalhar em equipe para desenvolver projetos que abordem problemas reais, promovendo a inovação e a solução de problemas complexos da engenharia. A disciplina é fundamental para a formação de engenheiros mecânicos capacitados e preparados para enfrentar os desafios do mercado de trabalho.

Aplicação na Extensão

A extensão universitária é uma das principais formas de interação entre a universidade e a comunidade. No contexto do Projeto Integrador I, a extensão permite que os alunos apliquem seus conhecimentos em projetos que beneficiem a sociedade. Estes projetos podem incluir a melhoria de processos industriais, desenvolvimento de novas tecnologias, projetos de sustentabilidade, entre outros.

A participação em projetos de extensão não só enriquece a formação dos alunos, mas também contribui para o desenvolvimento regional, oferecendo soluções inovadoras para problemas locais e promovendo o crescimento econômico e social da comunidade.

Além disso, a extensão proporciona aos alunos uma visão prática da engenharia, desenvolvendo habilidades como trabalho em equipe, comunicação, gestão de projetos e liderança. Estes projetos também permitem que os alunos vivenciem a realidade do mercado de trabalho, preparando-os melhor para suas futuras carreiras.

Projetos de Extensão

  • Construção de uma Retroescavadeira usando Seringas para Ensino Fundamental e Médio
  • Construção de um Veículo que Segue uma Linha utilizando Arduino

Construção de uma Retroescavadeira usando Seringas para Ensino Fundamental e Médio

Este projeto envolve a construção de uma miniatura de retroescavadeira utilizando seringas para simular o sistema hidráulico. É uma excelente ferramenta educacional para alunos do ensino fundamental e médio, pois permite que eles aprendam sobre princípios básicos de hidráulica e mecânica de forma prática e interativa.

Construção de um Veículo que Segue uma Linha utilizando Arduino

Neste projeto, os alunos desenvolvem um veículo autônomo que segue uma linha marcada no chão, utilizando uma plataforma Arduino para controle. Este projeto é ideal para introduzir conceitos de eletrônica, programação e automação, proporcionando uma experiência prática no desenvolvimento de sistemas embarcados.
